# 03 Configuración de clientes

## Windows
1. Configuración > Bluetooth y dispositivos > Impresoras y escáneres.
2. Agregar impresora.
3. Agregar manualmente.
4. Seleccionar impresora compartida por nombre.
5. Introducir la URL de CUPS.
6. Instalar el controlador.
7. Imprimir prueba.

## Ubuntu
1. Configuración > Impresoras.
2. Añadir impresora.
3. Elegir IPP.
4. Introducir URL.
5. Seleccionar controlador.
6. Imprimir prueba.

## Debian
Mismos pasos que Ubuntu.

## Linux Mint
1. Menú > Impresoras.
2. Añadir impresora de red.
3. Introducir URL IPP.
4. Seleccionar controlador.
5. Probar impresión.

## macOS
1. Ajustes del Sistema > Impresoras.
2. Añadir.
3. Pestaña IP.
4. Protocolo IPP.
5. Introducir URL.
6. Seleccionar controlador.
7. Imprimir prueba.

## Android
1. Misma red Wi-Fi.
2. Compartir > Imprimir.
3. Seleccionar impresora CUPS (o Mopria).

## Recomendaciones
- IP fija.
- Compartir impresora.
- Mantener CUPS actualizado.
