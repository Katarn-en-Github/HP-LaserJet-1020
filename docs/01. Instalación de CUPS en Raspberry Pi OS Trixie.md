# 01. Instalación de CUPS en Raspberry Pi OS Trixie

## Objetivo

Este documento describe cómo instalar y configurar **CUPS 2.4.x** sobre
una instalación limpia de **Raspberry Pi OS Trixie** en una **Raspberry
Pi Zero W**, dejándola lista para funcionar como servidor de impresión
en red.

> La instalación específica de la HP LaserJet 1020 se documenta en un
> segundo documento.

## 1. Requisitos previos

-   Raspberry Pi Zero W
-   Raspberry Pi OS Trixie
-   Acceso SSH
-   Internet

``` bash
sudo apt update
sudo apt full-upgrade -y
sudo reboot
```

## 2. Instalación de CUPS

``` bash
sudo apt install cups -y
sudo usermod -aG lpadmin tuusuarioenlapi
sudo systemctl enable cups
sudo systemctl start cups
```

## 3. Configuración de acceso a la red

Editar `/etc/cups/cupsd.conf`:

``` text
#Listen localhost:631
Port 631
Listen /run/cups/cups.sock
```

Agregar `Allow @LOCAL` en todos los bloques `<Location>`.

``` bash
sudo systemctl restart cups
sudo cupsctl --remote-admin --remote-any --share-printers
```

## 4. Inicio automático

Editar `/lib/systemd/system/cups.service` y agregar:

``` text
After=network.target network-online.target
Wants=network-online.target
```

Luego:

``` bash
sudo systemctl daemon-reload
sudo systemctl restart cups
```

## 5. Configuración de la impresora

Entrar a `http://IP_DE_LA_PI:631`, elegir **Administration → Add
Printer**, compartir la impresora e imprimir una página de prueba.

## 6. Clientes Windows y Linux

Usar la URL:

    http://IP_DE_LA_PI:631/printers/NOMBRE

## 7. Recomendación

Asignar una reserva DHCP en el router para mantener una IP fija.
